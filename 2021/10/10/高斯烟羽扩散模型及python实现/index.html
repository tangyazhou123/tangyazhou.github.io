    引入Arcpy

    Arcpy包含了很多处理地理数据的包，通过ArcGis的Python脚本窗口导入，也可以从软件自带的Python2.7 IDE导入。

    import Arcpy

    同时也可以引入一些python的数据处理库，如pandas

    import pandas as pd

    关于arcpy最全面的介绍，可以转到ArcGis的帮助文档，也是最好的学习工具。
